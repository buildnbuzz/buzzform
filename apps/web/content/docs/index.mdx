# Introduction

BuzzForm is a simple React form library for shadcn/ui. Declare fields, customize rendering, and get live validation with minimal boilerplate.

## Installation

A minimum version of Node.js 20 is required.

```bash tab="pnpm" selected
pnpm add @buildnbuzz/buzzform
```

```bash tab="npm"
npm install @buildnbuzz/buzzform
```

```bash tab="yarn"
yarn add @buildnbuzz/buzzform
```

```bash tab="bun"
bun add @buildnbuzz/buzzform
```
